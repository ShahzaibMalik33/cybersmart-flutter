🛡️ CyberSmart

CyberSmart is a Flutter-based cybersecurity awareness and phishing detection application designed to help users identify suspicious URLs, SMS messages, and files.

🚀 Features

- 🔗 URL Phishing Detection
- 📩 SMS Scam Detection
- 📁 File Security Scanning
- 📜 Scan History Tracking
- 🎨 Modern Dark UI
- 📱 Android Support
- ⚡ Fast and User-Friendly Interface

🔧 Installation

1. Clone the repository

```bash
git clone https://github.com/ShahzaibMalik33/cybersmart-flutter.git
```

2. Open the project

```bash
cd cybersmart-flutter
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the app

```bash
flutter run
```

# Purpose

The purpose of CyberSmart is to improve cybersecurity awareness and provide users with basic protection against phishing attacks and suspicious digital content.
